# Thuật toán sắp xếp

Thư mục này chứa các bài tập về thuật toán sắp xếp cơ bản trong C++.

## Nội dung chính
- `SelectionSort.cpp` — Sắp xếp lựa chọn (Selection Sort) cơ bản.
- `SelectionSort'.cpp` — Dấu vết (trace) thuật toán Selection Sort, in trạng thái mảng sau mỗi lần hoán đổi.
- `InsertionSort.cpp` — Insertion Sort cơ bản.
- `InsertionSort'.cpp` — Dấu vết (trace) thuật toán Insertion Sort, in trạng thái mảng trong từng bước.
- `QuickSort.cpp` — Thuật toán Quick Sort.

## Mô tả chung
- Các bài tập trong thư mục tập trung vào cách hoạt động của thuật toán sắp xếp và cách ghi lại quá trình hoán đổi.
- Một số bài yêu cầu in ra trạng thái mảng theo định dạng mẫu để dễ theo dõi thuật toán.

## Cách chạy
1. Mở file `.cpp` cần chạy.
2. Dùng lệnh g++:
   - `g++ -std=c++17 <file>.cpp -o <file>.exe`
3. Chạy chương trình:
   - `<file>.exe`

## Lưu ý
- Đọc kỹ đề bài để xác định đầu vào và định dạng xuất ra.
- Với bài trace, kết quả phải khớp chính xác theo mẫu ví dụ.
